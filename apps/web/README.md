# YesNo.Win Frontend

Next.js frontend for the prediction market platform.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- TailwindCSS 4
- shadcn/ui
- Thirdweb v5
- Zustand (state management)

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
pnpm start
```

## Features

- Market browsing and discovery
- Buy/sell shares with real-time pricing
- Portfolio management
- Admin dashboard for market creation and resolution
- Responsive design with modern UI
